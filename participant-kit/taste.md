# Visual Taste Rules

Use these rules for every chart.

## Overall style

- Static PNG/SVG.
- White background.
- Charcoal text, not pure black.
- Clean sans font.
- No decorative gradients, shadows, 3D, icons, or chartjunk.
- Let data and labels carry the design.

## classic dataviz craft-inspired rules

- Maximise useful data-ink.
- Remove non-data ink.
- Make comparison explicit: baseline, prior period, peer, target, or normal range.
- Keep visual effect proportional to data effect.
- Put words/numbers near the evidence they explain.

## Chart rules

- No legends if direct labels are possible.
- Prefer direct end labels for line charts.
- Remove top/right spines.
- Use minimal or no gridlines.
- Use colour sparingly: grey for context, one accent for focus.
- Avoid default matplotlib colour cycle.
- Avoid yellow/white low-contrast palettes.
- Format numbers humanly.
- Drop redundant axis titles.

## Anti-patterns

- Pie/donut unless part-to-whole is the only task.
- 3D charts.
- Rainbow palettes.
- Stacked bars when comparison matters.
- Generic “insightful trends” title.
- Unsupported causal language.
