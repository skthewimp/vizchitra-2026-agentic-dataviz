# Participant Kit - Agentic DataViz

Goal: build a small repeatable LLM-assisted dataviz workflow.

Work through files in order:

1. `context.md`
2. `taste.md`
3. `story_finding_prompt.md`
4. `visual_brief_template.md`
5. generate chart code/output
6. `chart_review_checklist.md`
7. `next_run.md`

Recommended rule: do not ask for a chart until you have a candidate story and visual brief.
